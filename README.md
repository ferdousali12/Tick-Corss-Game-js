# Tic-Tac-Toe Game

A classic Tic-Tac-Toe implementation built with vanilla JavaScript, HTML, and CSS.

## Features
- ✅ Two-player gameplay (O vs X)
- ✅ Win detection across all 8 patterns
- ✅ Turn-based logic with instant feedback
- ✅ Reset & New Game functionality
- ✅ Responsive UI

## How It Works
- Players take turns clicking boxes (O starts first)
- Game checks for winners after each move
- Boxes disable after being clicked
- Win message displays when 3-in-a-row achieved
- Reset button clears the board for a new round

## Tech Stack
- **HTML** - Game structure
- **CSS** - Styling & layout
- **JavaScript** - Game logic, event handling, win detection
